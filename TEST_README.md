# 测试页面使用说明

## 启动测试页面

运行以下命令启动测试页面:

```bash
npm run test
```

或者手动启动:

```bash
npm run dev
```

然后在浏览器中访问: `http://localhost:3000/test.html`

## 功能说明

测试页面提供了一个右侧的可折叠控制面板,包含以下功能:

### 1. 视图导航
快速切换到不同的游戏视图:
- **开场动画 (Intro)**: 游戏开场介绍
- **物品栏 (Inventory)**: 主要物品选择界面
- **录音机 (Recorder)**: 录音日志查看界面
- **笔记本 (Notebook)**: 博客笔记查看界面
- **胶片选择 (Film Select)**: 胶片选择和解锁界面

### 2. 当前状态显示
实时显示:
- 当前视图
- 已解锁日志数量
- 已解锁博客数量
- 已解锁胶片数量
- 录音机自动播放状态

### 3. 批量解锁控制
一键解锁所有内容:
- **解锁所有日志**: 解锁所有录音日志
- **解锁所有博客**: 解锁所有笔记本博客
- **解锁所有胶片**: 解锁所有胶片内容
- **重置所有进度**: 重置到初始状态

### 4. 单项解锁
通过展开列表,可以单独勾选/取消勾选:
- 日志列表中的每一条日志
- 博客列表中的每一篇博客
- 胶片列表中的每一个胶片

### 5. 其他控制
- **切换录音机自动播放**: 控制录音机是否已经自动播放过

## 使用场景

### 测试特定功能
1. 点击右侧面板的视图按钮直接跳转到对应界面
2. 无需按照游戏流程逐步解锁

### 测试解锁状态
1. 使用批量解锁按钮快速解锁所有内容
2. 或使用单项勾选测试特定解锁状态

### 测试边界情况
1. 测试空状态(重置所有进度)
2. 测试满状态(解锁所有内容)
3. 测试部分解锁状态(勾选部分内容)

### 调试特定视图
1. 直接跳转到需要调试的视图
2. 设置所需的解锁状态
3. 观察界面表现

## 面板控制

- 点击左侧的 **→** / **←** 按钮可以显示/隐藏测试面板
- 面板会覆盖在游戏界面右侧,不影响游戏主体显示
- 面板内容可滚动,支持查看所有测试选项

## 注意事项

- 测试页面与正式游戏使用相同的组件,确保测试结果的准确性
- 状态更改会立即生效,可以实时观察效果
- 测试面板不会出现在正式游戏中(使用 `index.html` 启动)

## 正式游戏启动

如需启动正式游戏(无测试面板):

```bash
npm run dev
```

然后访问: `http://localhost:3000/`
